### Site moderators

Roles:

- **VocaDB**: https://vocadb.net/User?groupId=Moderator
- **UtaiteDB**: https://utaitedb.net/User?groupId=Moderator
- **TouhouDB**: https://touhoudb.com/User?groupId=Moderator

Responsibilities:

- Enforcing website rules and guidelines
- Trusted user promotions
- Wiki management
- Handling artist verifications
- Handling content removal requests
- Maintaining tag mappings (Niconico tag -> VocaDB tag)
- Handling spammers (with [StopForumSpam](https://www.stopforumspam.com/))
- Handling DMCA, GDPR, user account deletions, etc.
