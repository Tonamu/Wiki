[Event entries](https://vocadb.net/Event) are for physical or digital voice synth related events.

Event entries can be created from: https://vocadb.net/Event/Edit

A full list of event related rules is available [here](/rules/events)

## Event types

There are two types of events:

1) Series events: Event is a part of an [event series](/docs/secondary-entry-types/event-series)

- Series name and event category are inherited to reduce duplicate information.

2) Standalone events

## Event categories

<RuleEmbed ruleId={97} fields={["id"]}/>

### Album release fair

(AKA. doujin fair): For conventions with the primary intention of selling albums. This includes Comiket, The Voc@loid M@ster and M3.

### Character anniversary

Character birthdays (release dates).

### Club

DJ party events.

### Concert

Music concerts such as [Magical Mirai](https://vocadb.net/Es/81/hatsune-miku-magical-mirai).

A song list with a [concert setlist](https://wiki.vocadb.net/docs/secondary-entry-types/song-lists#1-concert-setlists) category can be connected to any event entry.

- Most of the information about the concert should go to the event entry.

### Contest

Song contests such as [ProSeka NEXT](https://vocadb.net/Es/260/proseka-next).

### Festival

Song posting festivals such as [Song Title Unification Festival](https://vocadb.net/Es/310/### umei-touitsu-matsuri).

### Convention

For generic conventions such as [Nico Nico Chokaigi](https://vocadb.net/Es/77/nico-nico-chokaigi).

### Other

Events that do not fit into any of the categories above.

Events can be [tagged](/docs/secondary-entry-types/tags) and event participants can be credited through [artist entries](/docs/artists/artist-entry-page).
