Venue is an [entry type](/docs/other-guidelines/entry-types) for managing details for the (physical) [event](/docs/secondary-entry-types/events) location.

Venue entries can be created from: https://vocadb.net/Venue/Edit

Venue data fields include:

1) Generic entry fields:

- Names/Links/Description/etc.

2) Geographic information:

- Coordinates
- Country/Region
- Address
