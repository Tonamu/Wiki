Follow the [romanization guidelines](/docs/romanization-guidelines) when romanizing titles.

For Japanese, use the rules of modified [Hepburn romanization](https://en.wikipedia.org/wiki/Hepburn_romanization) unless an exception has been mentioned.

For Chinese, use the [Pinyin system](https://en.wikipedia.org/wiki/Pinyin).
